# Blood Donor Management System

## Overview

Blood Donor Management System is a web-based application developed to simplify the management of blood donor data, blood stock, and donation transactions. This project was created as part of a university web programming project.

## Features

- Admin Login
- Donor Data Management (CRUD)
- Blood Donation Transactions
- Blood Stock Management
- Blood Donation Reports

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Installation

1. Download or clone this repository.
2. Import `dbdonordarah.sql` into phpMyAdmin.
3. Configure the database connection in `koneksi.php`.
4. Run the project using XAMPP or Laragon.
